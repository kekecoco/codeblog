### 0X01 Java的基本程序设计结构
1. 数据类型:
    + 从Java7开始,加上前缀0b或0B就可以写二进制数.
    + 在Java中,所有的数值类型所占据的字节数量与平台无关.
    + Java中没有任何无符号形式的int,long,short或byte类型.
    + 整数:
        + byte: 1字节
        + short: 2字节
        + int: 4字节
        + long: 8字节
    + 浮点类型:
        + float: 4字节
        + double: 8字节
        + 浮点数不适用于无法接受舍入误差的金融计算中.
    + boolean类型:
        + 整型值和布尔值之间不能进行相互转换.
